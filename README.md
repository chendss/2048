# 原生js实现2048小游戏
## 详情
  * 利用了css3实现动画效果
  * 使用了语义化的html使得代码更可读，利用了部分html5特性。
  * 纯原生的javascript代码操控DOM控制元素组件
  * 大量使用es6语法使得代码更简洁
