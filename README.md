# 原生 js 实现 2048 小游戏

## 详情

-   利用了 css3 实现动画效果
-   使用了语义化的 html 使得代码更可读，利用了部分 html5 特性。
-   纯原生的 javascript 代码操控 DOM 控制元素组件
-   大量使用 es6 语法使得代码更简洁
